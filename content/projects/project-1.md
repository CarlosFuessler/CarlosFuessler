# Win95 OS Desktop

An interactive Windows 95-themed desktop environment running in the browser.

## Tech Stack

- Rust (compiled to WebAssembly)
- Custom CSS (Windows 95 design system)
- Minimal JavaScript glue

## Features

- Full window manager with drag, resize, minimize, maximize
- File manager with virtual filesystem
- Markdown viewer
- Terminal emulator
- Projects gallery

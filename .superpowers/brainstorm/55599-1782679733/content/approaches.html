Why A Wins

  • Most impressive for job interviews - memorable
  • Rust WASM is a natural fit for a window manager
  • Reusable window system for all apps
  • File manager, terminal, notepad are all windowed apps

Tradeoffs

  • Most complex to build
  • More Rust code for drag/resize/z-order
